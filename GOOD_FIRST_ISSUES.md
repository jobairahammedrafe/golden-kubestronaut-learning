# Good First Issues

Welcome to the Golden Kubestronaut Learning Path! This document contains beginner-friendly issues to help you get started with contributing.

## How to Contribute

1. Pick an issue from the list below
2. Comment on the GitHub issue (if exists) to let others know you're working on it
3. Fork the repository and create a branch
4. Submit a Pull Request when ready

---
