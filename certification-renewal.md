## üîÑ Renewal Process

### Step 1: Check Expiration Date
```bash
# Check your certification status on CNCF website
# Visit: https://www.cncf.io/certification/verify/
```

### Step 2: Schedule Renewal Exam
- Schedule your exam **30-60 days** before expiration
- Use the same PSI/LearnK8s portal
- Select the same certification exam

### Step 3: Prepare for Renewal
- Review latest Kubernetes release notes
- Practice with updated kubectl commands
- Complete mock exams

### Step 4: Take Renewal Exam
- Complete the exam before expiration date
- No grace period for expired certifications

2. **Practice Regularly**: Maintain hands-on skills
